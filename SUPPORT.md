# Support

Use the repository issue tracker for general support unless the matter is security-sensitive.

## Where to Ask

- Bug reports: open a bug report issue
- Feature requests: open a feature request issue
- Usage questions: open a discussion or issue in the repository, depending on what the host supports
- Security issues: follow [SECURITY.md](./SECURITY.md) and avoid public disclosure

## What to Include

- Browser version
- Extension version
- The GitHub page type involved, such as repository or profile
- Steps to reproduce
- Screenshots or copied report output when relevant
